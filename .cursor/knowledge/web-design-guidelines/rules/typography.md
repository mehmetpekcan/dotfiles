# Typography

- Prefer true ellipsis character over three periods for copy where style guidelines require it.
- Prefer typographic quotes over straight quotes where project conventions allow.
- Use non-breaking spacing for units and key labels where needed.
- Loading states should end with ellipsis (for example, "Loading...").
- Use tabular numerals for aligned number columns/comparisons when supported.
- Use balanced heading wrapping to prevent widows where supported.
